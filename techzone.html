techzone

Overview

TechZone was built to help users compare PC parts and make better purchasing decisions. The product combines live pricing, category browsing, and personalized tracking tools in a single, responsive interface.

Problem

PC component shoppers often switch between multiple sites to compare prices, compatibility information, and build recommendations.

Solution

A centralized dashboard with structured categories, searchable listings, and a watchlist workflow that surfaces useful data quickly.

My Contributions

Front-end implementation, responsive layout work, interface styling, and UX refinements for navigation clarity and component readability.

Technical Focus

API integration, consistent card/table UI patterns, and responsive behavior across desktop and mobile breakpoints.

Key Features

  1. Real-time component pricing pulled from external APIs.
  2. Build guide browsing with categorized hardware options.
  3. Personalized watchlist to track products over time.
  4. Responsive layouts for browsing and comparison on mobile devices.
